# Responsible Use

1984 is intended as a transparent, local-only corporate productivity tracking tool.

Use this software only where it is lawful and appropriate, and only with clear organizational policy and employee notice.

This project intentionally avoids:

- keylogging
- credential collection
- page content scraping
- OCR
- remote control
- live monitoring
- cloud synchronization
- covert spyware behavior

Operators are responsible for compliance with applicable labor, privacy, data protection, and information security requirements.
